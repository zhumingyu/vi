### Insert at the beginning of line

Change this text:

```javascript
var foo = 1
var bar = 'a'
var foobar = foo + bar
```

to this:

```javascript
;var foo = 1
;var bar = 'a'
;var foobar = foo + bar
```

Use `I` to insert at the beginning of line.

Add ';' to the beginning of these lines using `I`!
